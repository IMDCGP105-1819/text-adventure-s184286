Remembrance -- Help

Remembrance is an interactive story.  Each page is a scene.  At the bottom of each page is an action list and an action button.  Once you've read the scene select an action from the action list and then click the "Try Action" button. You will either be prompted for more input or given more information that will help you make your way through the story.  Some pages simply have a "Continue" button at the bottom.  Just click this button to move on, no further action is required. The action "puzzles" are not difficult for the emphasis is placed more on the story.  However, should you get stuck somewhere along the way the walk through will list all the action steps required to make it to the end.

In order to work your way through Remembrance you will need Netscape version 3.0 or higher or Internet Explorer version 4.01 or higher.  The reason for this is that an external JavaScript file is used and only Netscape 3.0 + and IE 4.01 + can handle this feature.  If you are experiencing JavaScript errors try downloading the latest version of Netscape or Internet Explorer.  You can get them for free from www.netscape.com and www.microsoft.com respectively.

If you find you don't have enough time to complete Remembrance in one sitting you can save your position by simply bookmarking the page you are currently viewing.  Then, if you wish to continue later, simply select your last scene from your bookmark list.

If you're ready to begin, click start.  Enjoy.
